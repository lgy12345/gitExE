1.进入要管理的文件夹
2.初始化

git init

git status(检查当前文件夹里面文件的状态)。然后git add 要管理的文件名称 （git add . 添加管理所有文件）

提交版本git commit -m "自定义名字"。之后重新执行git status 会变绿new file。 修改后执行git status又会显示红色modified。此时需要重新提交

git log (查看提交的所有版本)

回滚 ：git reset --hard (commit 代码号)

重新回滚：git reset --hard 6beef98

回到为暂存的状态（git add ）git reset HEAD (文件名)

git checkout -- (文件名) 再回到上一级状态
